# Пакет для работы с файлами (чтение/запись)

### Общая структура:
- Абстрактные классы — реализация заполнения коллекций элементов любого типа:
- Handler — взаимодействие с пользователем, определение способа ввода и длины массива элементов.
- Три других абстрактных класса — обработка заполнения коллекции элементов выбранным способом.

Конкретные классы являются реализацией абстрактных непосредственно для объектов класса `Student`.

---

### Цель:
Сделать три способа заполнения коллекции объектов и встроить в них проверку корректности данных.
"Пользователь должен иметь возможность выбирать варианты заполнения исходного массива данных (из файла, рандом, вручную) и его длину.".

В рамках него нужно:
- Продумать общий интерфейс для «источника данных».
- Реализовать ввод вручную.
- Реализовать чтение из файла.
- Реализовать случайную генерацию.

```java
class Student {
	int groupNumber;
	double averageGrade;
	String recordBookNumber;
}
```
